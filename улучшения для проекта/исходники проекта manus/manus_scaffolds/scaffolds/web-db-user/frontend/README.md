# Frontend for Web-DB-User Template

This is the frontend part of the web-db-user template, built with Vite, React, TypeScript, and TailwindCSS.

## Getting Started

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Build for production:
    ```bash
    npm run build
    ```
